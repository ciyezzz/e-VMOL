VMOL

(e-Video Mail OnLine)

Apa itu VMOL?

  • Sebuah Aplikasi pengiriman Surat/Kartu/Undangan/Advertising disertai Video berbasis Online/Mobile
  • Sebuah Generasi tren masa kini dalam pembuatan surat/kartu/undangan/Advertising

Sejarah VMOL

  • Dibangun oleh 2 orang : Michael(Founder) dan Septian Rahman(Co-Founder)
  • Ditemukan saat sedang berdiskusi mengenai project kolaborasi
  • Awalnya, direncanakan akan membuat Forum dengan Engine Chatting Online

Mengenal VMOL

  • 1. Setiap User yang merupakan member VMOL akan terbagi menjadi 2 : Paid User & Free User
    • Free User : format dan template design yang terbatas serta adanya pembatasan / limit terhadap jumlah kartu yang dapat dikirim (maksimal 50)
    • Paid User : format dan template design yang banyak serta tidak ada pembatasan / unlimitted terhadap jumlah kartu yang dapat dikirim
  • 2. User akan diberikan pilihan untuk mau atau tidak menambahkan video ke dalam design mereka

Lanjutan...

  • 3. Untuk video juga terdapat pembagian pada tiap user:
    • Paid User : Limit Durasi Video tidak ada, Limit Size Upload Video tidak ada, Format Video Bebas
    • Free User : Limit Durasi Video 5 menit, Limit Size Upload Video 25 MB, Format Video Bebas
  • 4. Semua User diberikan fasilitas Chatting Online, Integrasi dengan Social App lain seperti : Instagram, Facebook, Twitter, dll

Kelebihan VMOL

  • Efisien
  • Harga Ekonomis
  • Modern-Fashion Design Interface
  • Memperluas Jaringan Sosial
  • Terobosan Baru Social Media

Efisien

  • Tidak perlu keahlian khusus
  • Tersedia e-book cara penggunaan e-VMOL baik untuk free user maupun paid user
  • Mempermudah koneksi antar pengguna

Perbandingan VMOL dengan aplikasi lain

  • Menyediakan layanan yang cukup memuaskan untuk free user
  • Harga paid user yang terjangkau dibandingkan aplikasi lain (tersedia dalam weekly, monthly, yearly)
