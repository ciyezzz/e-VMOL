VMOL

(e-Video Mail OnLine)

Apa itu VMOL?

  • Sebuah Aplikasi masa kini dalam pembuatan dan pengiriman Surat/Kartu/Undangan/Advertising disertai Video berbasis Online/Mobile

Sejarah VMOL

  • Dibangun oleh 2 orang : Michael (Founder) dan Septian Rahman (Co-Founder)
  • Ditemukan saat sedang berdiskusi mengenai project kolaborasi.
  • Awalnya, direncanakan akan membuat Forum dengan Engine Chatting Online.

Mengenal VMOL

  • 1. Terbagi 2 : Paid User & Free User
  • 2. User akan diberikan pilihan apakah User mau atau tidak menambahkan video ke dalam design mereka.
  • 3. Semua User diberikan fasilitas Chatting Online, Integrasi dengan Social App lain seperti : Instagram, Facebook, Twitter, dll.

Perbedaan Free User & Paid User

  • 1. Dalam hal format dan template design

  • - Free User : Terbatas

  • - Paid User : Tidak terbatas

  • 2. Dalam hal limit

  • - Free User : Maksimal 50

  • - Paid User : Unlimitted

  • 3. Dalam hal durasi video, kapasitas video

  • - Free User : Maksimal 5 menit, Maksimal 25 MB

  • - Paid User : Unlimitted

  • 4. Dalam hal fasilitas tambahan

  • - Free User : Tidak ada layanan untuk merekam percakapan dari video call

  • - Paid User : Diberikan layanan untuk merekam percakapan dari video call (contoh : Skype) dan menguploadnya

Mengapa Efisien?

  • Tidak perlu keahlian khusus dalam mengoperasikannya.
  • Tersedia e-book cara penggunaan VMOL baik untuk free user maupun paid user.
  • Mempermudah koneksi antar pengguna.

Mengapa disebut Harga Ekonomis?

  • Harga yang cukup terjangkau dimana < IDR 500 ribu (per bulan).
  • Discount bagi user lama (dalam hal perpanjangan account).
  • Discount bagi user yang merekomendasikan VMOL kepada teman.
  • Discount bagi user yang langsung memperpanjang account selama 1 tahun.
  • Pembayaran bisa menggunakan PayPal, BitCoin, Visa, MasterCard.

Mengapa disebut Modern-Fashion Design Interface?

  • Tampilan antarmuka pengguna yang futuristik.
  • User akan merasakan sensasi Social Media yang benar-benar beda daripada biasanya.

Mengapa memiliki Modern-Fashion Design Interface?

  • VMOL memiliki design interface yang modern namun simple sehingga tidak membuat para user merasa jenuh.
  • Free User dapat mengubah warna tampilan interface sesuai dengan warna yang disediakan VMOL.
  • Paid User dapat mengubah warna tampilan interface dengan jenis warna yang lebih bervariasi atau pun user ingin mengupload gambar sendiri untuk dijadikan background interface.
  • Tersedia berbagai tema yang menarik untuk user interface, baik gratis maupun berbayar.

Mengapa dapat Memperluas Jaringan Sosial?

  • Melalui video, orang akan lebih mudah mengenali orang lain dibandingkan dengan melihat gambar atau profile picture.
  • Melalui video juga, dapat mengetahui orang yang memiliki hobi yang sama sehingga dapat tercipta suatu komunitas/kumpulan orang-orang yang memiliki hobi yang sama.
  • Adanya fasilitas chatting online sehingga user dapat lebih mengenal satu sama lain.

Mengapa disebut Terobosan Baru Social Media?

  • Merupakan terobosan baru (pertama kali di dunia) dalam pembuatan Surat/Kartu/Undangan/Advertising.
  • Interaksi di social media pada umumnya masih menggunakan media text maupun foto (media video masih jarang).
  • Video merupakan media yang lebih lengkap karena terdiri dari suara, gambar, dan tulisan.

Mengapa Teknologi Streaming Video di VMOL Disebut Memukau?

  • Pencegahan Buffering dengan Teknologi Anti-Buffering sehingga memberi kenyamanan bagi user.
  • Paid User dapat mengupload / mendownload video dengan cepat.
  • Paid User dapat sambil menonton video yang sedang didownload
  • Paid User dapat meningkatkan kualitas video yang akan dilihatnya sehingga tampilan video lebih realistik.

Perbandingan VMOL dengan Aplikasi Lain

  • New user dapat langsung memakainya (menjadi free user) dimana pada aplikasi lain, new user hanya dapat memakainya pada masa trial (terbatas).
  • Menyediakan layanan yang sangat memuaskan bahkan untuk free user sekalipun.
  • Harga paid user yang terjangkau dibandingkan aplikasi lain (tersedia dalam weekly, monthly, yearly).
  • Customer service 24 jam yang siap melayani pertanyaan, keluhan maupun saran dari user.
